Comment se connecter via QR Code ?
  1. Ouvrez WhatsApp sur votre téléphone.
  2. Allez dans Paramètres > Appareils connectés > Lier un appareil.
  3. Orientez votre caméra vers l'écran pour scanner le code QR ci-dessous.
  4. Une fois scanné, la connexion est automatique !

Scannez pour connecter WhatsApp ou liez via numéro de téléphone

Comment lier via numéro de téléphone ?
  1. Entrez votre numéro (ex: 243...) dans le champ ci-dessous et cliquez sur "Lier via numéro de téléphone".
  2. Un code s'affiche.
  3. Ouvrez votre WhatsApp sur le téléphone.
  4. Allez dans Paramètres > Appareils connectés > Lier un appareil.
  5. Au lieu de scanner, cliquez en bas sur "Lier avec le numéro de téléphone à la place".
  6. Saisissez le code affiché sur le site.
  7. C'est fait ! Le serveur est maintenant connecté.
